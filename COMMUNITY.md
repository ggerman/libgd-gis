# libgd-gis Community

libgd-gis is a native GIS and map rendering engine for Ruby.  
It provides map projections, layers, icons, labels, and tile-grade raster output built on top of ruby-libgd.

This community exists to:
- help people render real maps with Ruby
- share datasets, scripts, and examples
- guide the roadmap of Ruby-based GIS
